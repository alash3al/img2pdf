# img2pdf
convert a directory of images to a PDF file

# installation
* You must have `Golang` compiler
* open your `terminal` and run `go get -u github.com/alash3al/img2pdf`
* have fun `img2pdf -src /path/to/images-dir -as ./images.pdf`
